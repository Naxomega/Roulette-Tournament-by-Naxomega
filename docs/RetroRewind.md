# COURSES

## Conditions de victoire

- Standard (Points)
- Position précise (1-12e)
- Mode Elimination
- Position moyenne (1-12e en moyenne sur les courses)
- Position globale (Equipe uniquement)

## Modificateurs

- Aucun
- Equipe (2 Equipes)
- Duo (Equipes de 2)
- Trio (Equipes de 3)
- Quatuor (Equipes de 4)
- Compo spécifique (Choix du personnage et du kart/moto pour tous les participants (une composition pour tout le monde))
- Type de véhicule (Kart, Moto)
- Type de Personnage (Léger, Moyen ou Lourd)
- 200cc
- Objets (Aléatoire Géneral (Activation ou désactivation d'objets choisis aléatoirement), Obtention Aléatoire (tout le monde peut obtenir tout les objets), Mode Chaos (Plus d'explosions !!), Pluie d'Objets (Pleins d'objets tombent sur la piste)
- Circuits (Vanilla (Ceux du jeu de base), Retro (Jeu de base + Ceux des autres Mario Kart), Custom (Circuits crées par la communauté)
- Boîte à objets rapide (Fast Item) (Les boîtes réapparaissent plus vite)
- Nuage Zap (Nuage Shock) (Remplace le Nuage Méga (Provient de RR) par le Nuage Zap (Présent de base dans le jeu), appliquant l'effet d'un éclair à l'échéance de l'objet
- Tout les objets tombent (Falling Items) (Vous faites tomber TOUS vos objets sur la piste quand vous percutez un Bill, une bombe ou autre obstacle vous reversant (Carapace Bleue par exemple...)

# INFOS SUPPLEMENTAIRES

## Condition "Position Précise"

L'objectif de cette position est de terminer la manche dans la position definie au début du round (2e, 4e...), les points du classement de round seront attribues de la méme manieére que le jeu de base.

Exemple :  
- Position définie : 15pts  
- 1 d'écart : 12pts  
- 2 d'écart : 10pts  

---

## Condition "Position Moyenne"

La Position finale des participants est calculée en faisant la moyenne de leurs positions au cours des différentes manches, qui est arrondie a l'entier inférieur.

Exemple :  
3e, 4e, 2e, 1er → 2.5e → 2e

---

## Condition "Position Globale"

La position de l'equipe est calculée en faisant la moyenne de la position de tous les participants.

Ces résultats s'additionnent et l'equipe qui a le moins de points (moins car c'est les positions, ou un chiffre moins élevé gagne : 1er > 4e) remporte la manche.

---

# Classement général

Une fois le classement des rounds fait, chaque personne remporte au classement general le nombre de points d'une course standard :

- 1er : 15pts  
- 2e : 12pts  
- etc.

Ces points sont définis par leur classement dans ce round (et de ces régles).

Pour ce qui concerne le mode en equipe :

- Membres de l'equipe gagnante : **10pts**
- Membres de l'equipe perdante : **5pts**
